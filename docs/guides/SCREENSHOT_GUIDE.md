# 📸 スクリーンショット撮影ガイド

DNS OSINT Pro ver2.0のREADME用スクリーンショット撮影手順です。

---

## 📋 撮影が必要なスクリーンショット

### 1. main.png - メイン画面
**内容:** DNS情報、WHOIS、サジェスト分析の全体表示

**撮影手順:**
1. Chrome拡張機能を開く
2. 適当なドメイン（例: `example.com`）を入力して検索
3. すべての情報が表示されたら、ポップアップ全体をスクリーンショット
4. `docs/screenshots/main.png` として保存

**推奨サイズ:** 800x600px

---

### 2. site-health.png - サイト健康診断
**内容:** りんく、こん太、たぬ姉によるサイト診断結果

**撮影手順:**
1. 任意のWebサイト（例: 実在する企業サイト）を開く
2. Chrome拡張機能を開く
3. 「💫 君斗りんくのWEBサイト健康診断」セクションが表示されるまで待つ
4. 診断結果（良好な点・改善点・問題点）が表示されたら、そのセクションをスクリーンショット

**撮影のポイント:**
- りんく、こん太、たぬ姉のキャラクターアイコンが見えるようにする
- 診断結果の内容が読めるようにする
- 色分け（緑・オレンジ・赤）が分かるようにする

**推奨サイズ:** 700x500px

---

### 3. suggest-check.png - サジェスト風評チェック（正常）
**内容:** サジェストが正常な場合の表示

**撮影手順:**
1. 有名企業や問題のないサイトを開く（例: Google.com、Amazon.co.jp）
2. Chrome拡張機能を開く
3. サジェスト情報が表示されるまで待つ
4. **ネガティブキーワードが含まれていない**サジェスト結果をスクリーンショット

**撮影のポイント:**
- Google、Yahoo!、Bingのサジェストすべてが表示されていること
- リンクアイコン（🔗）が見えること
- 口コミサイトリンクボタンが見えること

**推奨サイズ:** 700x600px

---

### 4. reputation-alert.png - 風評被害検出アラート
**内容:** ネガティブキーワードを検出した場合の警告表示

**撮影手順:**
1. 風評被害がある（またはありそうな）企業サイトを開く
2. Chrome拡張機能を開く
3. サジェスト情報を取得
4. **赤色の警告が表示された**場合、そのセクション全体をスクリーンショット

**撮影のポイント:**
- 赤色の警告枠（⚠️ 風評被害の可能性が検出されました）が見えること
- 赤字でハイライトされたネガティブキーワードが見えること
- 「📱 無料でLINE相談する」ボタンが見えること

**推奨サイズ:** 700x600px

**注意:**
- 実在する企業の風評被害情報は公開しないこと
- デモ用に加工するか、架空のサンプルを使用すること
- または、テスト用のダミーサイトで撮影すること

---

### 5. review-links.png - 口コミサイトリンク
**内容:** 口コミサイトへのリンクボタン表示

**撮影手順:**
1. 任意のサイトを開いてChrome拡張機能を起動
2. サジェストセクションまでスクロール
3. 「💬 口コミサイトで評判をチェック」セクションを拡大表示
4. すべてのボタン（Googleマップ、転職会議、OpenWorkなど）が見えるようにスクリーンショット

**撮影のポイント:**
- すべてのボタンが見えること
- ホバー時の色変化も撮影できればベター

**推奨サイズ:** 600x400px

## 🛠️ スクリーンショット撮影のツール

### Windows
- **Snipping Tool** (標準搭載)
  - Windows + Shift + S
- **Screenpresso** (無料)
  - https://www.screenpresso.com/

### Mac
- **Command + Shift + 4**（標準機能）

### Chrome拡張機能
- **Awesome Screenshot**
  - ポップアップ全体を簡単に撮影可能

## 📐 画像の編集・加工

### 推奨ツール
- **Photoshop**（有料）
- **GIMP**（無料）
- **Canva**（無料・オンライン）

### 編集内容
1. **サイズ調整**
   - 幅: 600-800px程度
   - 高さ: 400-600px程度

2. **個人情報のマスキング**
   - 実在する企業名を隠す場合はぼかし
   - IPアドレスなど

3. **画質の最適化**
   - PNG形式で保存
   - ファイルサイズ: 500KB以下推奨

## 📝 注意事項

1. **個人情報の保護**
   - 実在する企業の風評情報は掲載しない
   - IPアドレス、メールアドレスなどはマスキングする

2. **著作権**
   - 他社のロゴやコンテンツが含まれる場合は注意
   - デモ用のダミーデータを使用推奨

3. **画質**
   - ぼやけた画像は避ける
   - 文字が読めるサイズで撮影

4. **統一感**
   - すべてのスクリーンショットを同じブラウザで撮影
   - 同じズーム倍率で撮影
