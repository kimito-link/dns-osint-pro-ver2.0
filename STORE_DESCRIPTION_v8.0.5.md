# Chrome Web Store 説明文 - v8.0.5（サジェスト汚染チェッカー）

## 短い説明（132文字以内）
複数の検索エンジンのサジェスト機能から、ネガティブキーワードを自動検出。DNS/WHOIS/SSL診断、逆SEO・口コミサイトチェック、CSV出力機能を搭載。

## 詳細な説明

ドメインのDNS情報、SSL、メールセキュリティ、サジェスト汚染・風評被害をワンクリック診断。SEO問題も検出します。

【v8.0.5 サジェスト汚染チェッカー アップデート】

✨ 新機能
- **逆SEO・ネガティブサイトチェック**: 5ch、爆サイ、jpnumber、ホスラブへのワンクリック検索リンクを追加。個人名・会社名でネガティブな投稿が表示されていないか確認可能
- **リブランディング**: サジェスト汚染チェッカーとして生まれ変わりました

🎨 改善
- **風評関係を最優先表示**: 風評被害チェック・個人名ネガティブチェックを最上部に表示
- **口コミ相談リンク**: リバースハック相談（口コミ）ボタンを追加
- **リバースハック鮫ロゴ**: ヘッダー・ファビコンを統一

---

【主な機能】

🔍 包括的なドメイン診断
- DNS情報の取得: A、AAAA、MX、TXT、NS、CNAME、SOAレコードなど
- WHOIS情報の取得: RDAP対応で最新の情報を取得
- SSL証明書の確認: 証明書の有効期限や発行者を確認
- メールセキュリティ診断: SPF、DKIM、DMARCの設定をチェック

🛡️ セキュリティチェック
- WordPress・PHPバージョンチェック: 古いバージョンの警告表示
- プラグイン脆弱性チェック: セキュリティリスクの可視化
- Contact Form 7バージョンチェック: 脆弱性の早期発見
- マルウェア検出: Google Safe Browsing APIによる安全性チェック
- ドメイン期限チェック: ドメインの有効期限を監視
- 専門家への相談導線: 問題解決のサポート

📊 SEO診断機能
- Googleインデックス数の確認: サイトの検索エンジン登録状況
- サイトマップ解析: sitemap.xmlの構造を可視化
- サイト構造の可視化: ページ階層をツリー表示
- メタ情報の取得: タイトル、ディスクリプション、OGPタグをチェック

🚨 風評被害・サジェスト汚染対策
- ネガティブキーワード検出: 複数検索エンジンから自動検出
- Google/Yahoo!/Bingサジェスト汚染チェック
- 逆SEO・ネガティブサイトチェック: 5ch、爆サイ、jpnumber、ホスラブへのワンクリック検索
- 口コミ・企業評判サイトへのリンク: Googleマップ、転職会議、OpenWorkなど
- 個人名ネガティブチェック: 役員・スタッフの風評を調査

🌐 その他の便利機能
- Wayback Machineの履歴確認
- サブドメイン検索
- 右クリックメニューから素早くアクセス
- 結果のエクスポート: CSV形式でダウンロード
- コピー機能、レスポンシブデザイン対応
