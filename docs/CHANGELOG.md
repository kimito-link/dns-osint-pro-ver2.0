# 📝 変更履歴 (CHANGELOG)

## [6.5.1] - 2025-11-08

### 🎉 新機能
- **サイトタイトル常時表示**: 検索窓の下にサイト情報を表示
  - りんくの画像（45x45px）
  - タイトルを白背景のボックスで強調
  - 「このサイトのタイトルは [タイトル] だよ！」形式
- **デバッグログ機能**: コンソールログを統合表示
  - console.log/info/warn/errorを自動収集
  - SEO情報タブ内に配置
  - トグルボタンで表示/非表示切り替え
  - タイムスタンプ付きログ表示
  - コピーボタンでログを簡単にコピー

### ⚡ パフォーマンス改善
- **サイトマップ処理の高速化**: 約15倍の速度改善
  - 並列処理による最大10ページの同時タイトル取得
  - タイムアウトを3秒→2秒に短縮
  - 処理時間: 最大30秒→最大2秒

### 🐛 バグ修正
- **タブ切り替え問題を修正**
  - SEO情報タブから診断結果タブに戻れない問題を解決
  - data-tab属性を直接使用するように改善
  - イベント処理の最適化
- **CSPエラー解決**
  - Manifest V3のContent Security Policy完全準拠
  - インラインJavaScript（onclick属性）を削除
  - イベントリスナーを適切に実装

### 🎨 UI/UX改善
- サイトタイトル表示のデザイン調整（3回の反復改善）
- デバッグログの初期状態を非表示に変更
- トグルボタンの配置改善
