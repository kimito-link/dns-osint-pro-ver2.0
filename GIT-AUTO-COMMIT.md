# 🤖 Git自動コミットガイド

このプロジェクトには自動コミット機能が組み込まれています。

## 📋 使い方

### 方法1: PowerShellスクリプト（推奨）

```powershell
# カスタムメッセージでコミット
.\auto-commit.ps1 "v6.7.1: 見出しテキスト表示機能追加"

# デフォルトメッセージでコミット（タイムスタンプ付き）
.\auto-commit.ps1
```

### 方法2: Git Bash

```bash
# カスタムメッセージでコミット
./auto-commit.sh "v6.7.1: 見出しテキスト表示機能追加"

# デフォルトメッセージでコミット
./auto-commit.sh
```

### 方法3: VSCodeショートカットキー ⭐ 最も便利！

#### 設定手順

1. **VSCodeを開く**
2. `Ctrl + Shift + P` → "Preferences: Open Keyboard Shortcuts (JSON)" を選択
3. 以下を追加：

```json
[
  {
    "key": "ctrl+shift+c",
    "command": "workbench.action.tasks.runTask",
    "args": "🔄 自動コミット"
  },
  {
    "key": "ctrl+alt+c",
    "command": "workbench.action.tasks.runTask",
    "args": "⚡ クイックコミット（メッセージなし）"
  }
]
```

#### 使い方

- **`Ctrl + Shift + C`**: コミットメッセージを入力してコミット
- **`Ctrl + Alt + C`**: デフォルトメッセージで即座にコミット

---

## 🎯 自動化のメリット

✅ **コマンド不要**：ショートカットキー1つでコミット  
✅ **タイムスタンプ自動付与**：いつコミットしたか記録  
✅ **変更なしを自動検出**：無駄なコミットを防止  
✅ **作業効率UP**：コミット忘れを防止

---

## 📝 コミットメッセージのルール

### プレフィックス推奨

- `feat:` 新機能追加
- `fix:` バグ修正
- `docs:` ドキュメント更新
- `style:` UI/スタイル変更
- `refactor:` リファクタリング
- `test:` テスト追加
- `chore:` その他の変更

### 例

```
v6.7.1: 見出しテキスト表示機能追加
feat: H1-H6タグの実際のテキストを表示
fix: RDAP IPエラーを修正
docs: README更新
```

---

## ⚠️ 注意事項

- 大きな変更の前には手動でコミットすることを推奨
- プッシュは含まれていません（手動で `git push`）
- 変更がない場合は自動でスキップされます
