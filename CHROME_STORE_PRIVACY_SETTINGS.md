# Chrome Web Store プライバシー設定ページ - 入力内容

## 単一用途の説明

【単一目的】
Webサイトのドメインに関する公開情報（DNS、WHOIS、SSL、SEO情報、サジェスト汚染チェック、マルウェア検出、ドメイン期限チェックなど）を一括で取得・表示し、サイト管理者やマーケターの業務を効率化するツールです。すべての機能は「ドメイン情報の可視化とセキュリティ診断」という単一の目的に集約されています。

---

## 権限が必要な理由

### 1. activeTab が必要な理由

【activeTab権限の使用理由】
現在アクティブなタブのURL（ドメイン名）を取得するために使用しています。

具体的な用途：
- ユーザーが拡張機能を起動した際、現在のページのドメイン名を自動的に入力欄に表示
- 手動入力の手間を削減し、ユーザビリティを向上
- サイト健康診断機能で、現在のページのHTML構造、メタタグ、セキュリティヘッダーを分析

この権限により取得する情報：
この権限により取得する情報は、現在のタブのURLのみです。ページの内容やユーザーの行動は一切取得しません。

---

### 2. storage が必要な理由

【storage権限の使用理由】
ユーザーが設定したAPIキーや検索履歴をブラウザ内に保存するために使用しています。

具体的な用途：
- ユーザーが設定したAPIキー（SecurityTrails API、WHOIS XML API、Google Custom Search API、Google Maps APIなど）をブラウザ内に保存
- 過去に検索したドメインの履歴をブラウザ内に保存し、次回以降の検索を高速化
- ユーザーが設定したカスタマイズオプション（表示設定など）を保存
- Chrome Syncを使用して、ユーザーがログインしているChromeアカウントの端末間でAPIキーを同期（オプション設定で有効化した場合）

この権限により取得する情報：
- すべてのデータはお客様のブラウザ内（chrome.storage.local）にのみ保存されます
- APIキーはChrome Syncを使用して端末間で同期されますが、Googleのサーバー経由でのみ同期され、開発者のサーバーには送信されません
- 開発者や第三者がこれらのデータにアクセスすることは一切ありません
- データは暗号化されたブラウザのローカルストレージに保存され、他のWebサイトやアプリケーションからはアクセスできません

---

### 3. tabs が必要な理由

【tabs権限の使用理由】
現在開いているタブのドメイン名とURLを取得し、自動的にDNS/WHOIS情報を表示するために使用しています。

具体的な用途：
- 現在開いているタブのドメイン名とURLを取得し、自動的にDNS/WHOIS情報を表示
- 新しいタブで外部サービス（Google検索、Yahoo!検索、Bingなど）を開く機能を提供
- 右クリックメニューから選択したテキスト（ドメイン名など）を取得し、検索機能を提供

この権限により取得する情報：
- URL情報は機能の実行にのみ使用され、保存や送信は行いません（ユーザーが明示的に履歴として保存する場合を除く）
- タブの内容（閲覧履歴、入力フォームの内容など）は一切収集しません

---

### 4. contextMenus が必要な理由

【contextMenus権限の使用理由】
Webページ上で選択したテキスト（ドメイン名、IPアドレスなど）を右クリックメニューから素早く検索できる機能を提供するために使用しています。

具体的な用途：
- Webページ上で選択したテキスト（ドメイン名、IPアドレスなど）を右クリックメニューから素早く検索できる機能を提供
- ユーザーの操作性を向上させ、効率的なOSINT調査をサポート

この権限により取得する情報：
- 右クリックで選択されたテキストは、検索機能の実行にのみ使用されます
- 選択されたテキスト以外の情報（ページ全体の内容など）は一切収集しません

---

### 5. scripting が必要な理由

【scripting権限の使用理由】
サイト健康診断機能で、現在のページのHTML構造、メタタグ、セキュリティヘッダーを分析するために使用しています。

具体的な用途：
- サイト健康診断機能で、現在のページのHTML構造、メタタグ、セキュリティヘッダーを分析
- SEO情報の取得（タイトル、メタディスクリプション、OGPタグなど）
- WordPressバージョンやPHPバージョンの検出

この権限により取得する情報：
- ページのHTML構造とメタ情報のみを取得し、個人情報や入力フォームの内容は一切取得しません
- 取得した情報は、診断結果の表示にのみ使用され、外部に送信されることはありません

---

## ホスト権限について

本拡張機能は、以下の外部APIと通信するためにホスト権限を要求しています：

- Cloudflare DNS over HTTPS: DNS情報の照会
- RDAP.org: WHOIS情報の照会
- Archive.org (Wayback Machine): Webサイトの過去のスナップショット情報の取得
- crt.sh: SSL証明書の透明性ログからサブドメイン情報の取得
- Google Suggest API: Google検索のサジェスト情報の取得
- Yahoo! Suggest API: Yahoo!検索のサジェスト情報の取得
- Bing Suggest API: Bing検索のサジェスト情報の取得
- SecurityTrails API: 詳細なDNS履歴やサブドメイン情報の取得（APIキーが設定されている場合のみ）
- WHOIS XML API: 詳細なWHOIS情報の取得（APIキーが設定されている場合のみ）
- Google Custom Search API: Google検索結果の取得（APIキーが設定されている場合のみ）
- Google Maps API: 地図表示機能（APIキーが設定されている場合のみ）

送信されるデータ：
- 調査対象のドメイン名やIPアドレスのみが送信されます
- ユーザーの個人情報、閲覧履歴、入力内容などは一切送信されません
