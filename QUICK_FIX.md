# 🔧 エラー修正 - すぐに解決できます！

**エラー内容**: `The extensions gallery cannot be scripted`

---

## ❌ **問題の原因**

Chromeウェブストア（`chrome://extensions/`や拡張機能ページ）で拡張機能を開こうとしたため。

Chromeはセキュリティ上、以下のページで拡張機能の実行を禁止しています：
- `chrome://` で始まるページ
- `chrome-extension://` で始まるページ  
- Chrome ウェブストア

---

## ✅ **解決方法（30秒で解決！）**

### **ステップ1: 通常のWebサイトを開く**

以下のいずれかのサイトを開いてください：

```
推奨:
1. https://www.yahoo.co.jp/
2. https://www.google.com/
3. https://www.amazon.co.jp/
4. https://www.example.com/
```

### **ステップ2: 拡張機能を開く**

```
1. 開いたWebサイト上で拡張機能アイコンをクリック
2. ドメイン入力欄にドメイン名を入力（例: yahoo.co.jp）
3. 「検索」ボタンをクリック
```

### **ステップ3: アルファベット拡張をテスト**

```
4. サジェスト結果が表示されたら下にスクロール
5. 青いボタン「🚀 関連キーワードをさらに取得」を探す
6. クリック！
```

## 💡 **よくある質問**

### **Q: なぜChromeウェブストアで使えないの？**

**A**: セキュリティ上の理由です。

Chromeは、拡張機能が自分自身（Chromeの内部ページ）を改変できないようにしています。これにより、悪意のある拡張機能から保護されます。

### **Q: どのサイトなら使える？**

**A**: `https://` または `http://` で始まる通常のWebサイトすべて。

例:
- ✅ https://www.yahoo.co.jp/
- ✅ https://www.google.com/
- ✅ https://github.com/
- ✅ http://localhost:3000/

### **Q: 新しいタブページでも使える？**

**A**: いいえ、使えません。

`chrome://newtab/` も内部ページなので使用不可。
代わりに、任意のWebサイトを開いてください。
