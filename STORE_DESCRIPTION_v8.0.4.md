【使い方】

1. Chrome拡張機能をインストール
2. 診断したいサイトを開く
3. 拡張機能アイコンをクリック
4. 自動的にドメイン情報を取得・診断
5. 結果を確認し、必要に応じて対策

オプション設定（任意）:
- 拡張機能アイコンを右クリック → 「オプション」を選択
- Google APIキーを設定（カスタマイズ可能）

---

【プライバシー】

データ収集について:
- ユーザーが明示的に検索したドメイン情報のみを取得
- 個人を特定できる情報は一切収集しません
- データは外部に送信されません（公開API経由の情報取得のみ）

APIキーについて:
- オプション画面で設定したAPIキーはChrome Syncで端末間同期されます
- APIキーは安全に管理され、外部に送信されることはありません
